# Aurora-Siger

Aurora Siger 1.1 - Sistema de Telemetria e Decisão Este projeto implementa um sistema de telemetria e um algoritmo de decisão para a Missão Aurora Siger 
O sistema monitora cinco parâmetros críticos (temperatura interna/externa, integridade estrutural, níveis de energia, pressão dos tanques e status de módulos críticos) para determinar a viabilidade de uma decolagem, resultando em "PRONTO PARA DECOLAR" ou "DECOLAGEM ABORTADA".
ComandoPara executar o script de simulação:
bash
Telemetria: Definição e interpretação dos dados de telemetria provenientes dos sensores da nave.
Algoritmo de Verificação: Lógica de decisão baseada em faixas seguras predefinidas para cada parâmetro crítico.
Script em Python: Implementação do algoritmo de verificação, simulando a leitura de dados e exibindo o resultado da decisão.
Análise Energética: Cálculo da autonomia inicial da nave, considerando capacidade, carga atual, consumo estimado e perdas.
Riscos e Anomalias: Classificação de dados, identificação de anomalias e sugestões de mitigação de riscos.
